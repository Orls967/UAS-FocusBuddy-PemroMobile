FocusBuddy UAS Final Design System

Overview

FocusBuddy merupakan aplikasi produktivitas akademik yang dirancang untuk membantu mahasiswa mengelola tugas, meningkatkan fokus belajar, serta memonitor perkembangan produktivitas secara berkelanjutan.

Desain aplikasi mengusung pendekatan modern productivity application dengan tampilan yang bersih, hangat, profesional, dan mudah digunakan oleh mahasiswa.

Fokus utama desain adalah:

* Focus
* Productivity
* Academic Growth
* Simplicity
* Motivation

⸻

Design Principles

1. Focus First

Setiap halaman harus membantu pengguna menyelesaikan tujuan utama tanpa distraksi berlebihan.

Elemen penting selalu menjadi fokus visual utama.

Contoh:

* Timer menjadi fokus utama pada halaman Focus Session.
* Statistik menjadi fokus utama pada halaman Progress.
* Tugas menjadi fokus utama pada halaman Task Management.

⸻

2. Warm Academic Experience

Aplikasi tidak menggunakan gaya corporate atau enterprise.

Visual dirancang agar terasa:

* profesional
* ramah
* modern
* dekat dengan mahasiswa

⸻

3. Consistent Design Language

Seluruh halaman menggunakan:

* warna yang konsisten
* tipografi yang konsisten
* gaya kartu yang konsisten
* navigasi yang konsisten
* icon yang konsisten

Pengguna harus langsung mengenali identitas FocusBuddy di setiap halaman.

Screen Structure

1. Splash Screen

Tujuan:

Memperkenalkan identitas aplikasi sebelum masuk ke sistem.

Komponen:

* Logo
* FocusBuddy
* Tagline
* Loading Indicator

Catatan:

Seluruh elemen berada pada area visual tengah layar.

Tidak ditempatkan dekat status bar.

Tidak ditempatkan terlalu tinggi.

⸻

2. Login

Tujuan:

Masuk ke aplikasi.

Komponen:

* Email
* Password
* Login Button
* Sign Up Navigation

⸻

3. Register

Tujuan:

Membuat akun baru.

Komponen:

* Nama
* Email
* Password
* Confirm Password
* Register Button

⸻

4. Dashboard

Tujuan:

Memberikan gambaran produktivitas pengguna.

Komponen:

* Greeting
* Daily Focus Progress
* Current Streak
* Daily Momentum
* Motivational Quote
* Task Preview

⸻

5. Focus Timer

Tujuan:

Membantu pengguna menjalankan sesi fokus.

Komponen:

* Circular Timer
* Current Task
* Start Button
* Pause Button
* Session Summary

Timer menjadi elemen visual utama.

⸻

6. Task Management

Tujuan:

Mengelola seluruh tugas pengguna.

Komponen:

* Search
* Filter
* Task List
* Priority Indicator
* Add Task Button

Menerapkan konsep Browse.

⸻

7. Task Detail

Tujuan:

Menampilkan detail tugas.

Komponen:

* Task Information
* Progress
* Subtask
* Edit Button
* Delete Button

Menerapkan konsep Read.

⸻

8. Add/Edit Task

Tujuan:

Membuat atau mengubah tugas.

Komponen:

* Task Title
* Description
* Category
* Priority
* Save Button

Menerapkan konsep Add dan Edit.

⸻

9. Progress Analytics

Tujuan:

Menampilkan perkembangan produktivitas.

Komponen:

* Weekly Focus
* Monthly Trend
* Productivity Insight
* Personal Best Record
* Achievement Card

⸻

10. Profile & Settings

Tujuan:

Mengelola akun dan preferensi pengguna.

Komponen:

* User Information
* Focus Statistics
* Notification Settings
* Focus Settings
* Privacy Settings

⸻

Navigation

Menggunakan Bottom Navigation dengan lima menu utama:

1. Dashboard
2. Tasks
3. Timer
4. Progress
5. Profile

Aturan:

* Active State menggunakan Terracotta
* Inactive State menggunakan muted navy
* Seluruh halaman harus memiliki navigasi yang konsisten

⸻

Visual Consistency Rules

1. Semua CTA menggunakan warna Terracotta.
2. Semua indikator sukses menggunakan warna Green.
3. Semua kartu statistik utama menggunakan Navy.
4. Seluruh halaman menggunakan background Warm Background.
5. Border radius konsisten pada seluruh card dan button.
6. Spacing antar section harus konsisten.
7. Tidak menggunakan warna tambahan di luar design system.
8. Semua screen harus terlihat berasal dari satu produk yang sama.

⸻

Final Design Goal

FocusBuddy harus terasa sebagai aplikasi produktivitas akademik yang:

* modern
* profesional
* hangat
* memotivasi
* mudah digunakan

serta mampu membantu mahasiswa membangun kebiasaan belajar yang lebih terstruktur dan produktif.
